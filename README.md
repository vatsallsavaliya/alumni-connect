# Alumni Connect 🎓

A comprehensive Alumni Association platform for Government Engineering College.  
This web application bridges alumni with their alma mater, fostering networking, career opportunities, and institutional growth.  



## 🚀 Features

- **Alumni Registration & Login** – Easy and secure sign-up for alumni.  
- **Networking Hub** – Connect with peers by interests, departments, or profession.  
- **Job Portal** – Post and find career opportunities.  
- **Donations** – Secure channels for alumni contributions.  
- **Events & Reunions** – Manage and join alumni meetups and college events.  
- **Success Stories** – Showcase alumni achievements.  
- **Directory Search** – Find fellow alumni quickly.  
- **Feedback & Surveys** – Share insights to improve the alumni ecosystem.  
- **Admin Panel** – Manage users, jobs, events, content, and donations.  



## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** PHP  
- **Database:** MySQL  
- **Version Control:** Git & GitHub  



## 📂 Project Structure

## ⚙️ Installation & Setup

1. Clone the repository  
   ```bash
   git clone https://github.com/your-username/alumni-connect.git
Import the database

Create a MySQL database (e.g., alumni_platform)

Import the SQL file located in /database/

Configure database connection

Update DB credentials in config.php (hostname, username, password, dbname)

Run the project

Place project folder inside htdocs/ (XAMPP) or www/ (WAMP)

Start Apache & MySQL from XAMPP/WAMP

Open in browser:
http://localhost/alumni-connect/
